# sistema de tienda de ropa 

# Descripccion 
La tienda de ropa es una aplicación web donde se pueden ver diferentes prendas, elegir la talla y agregarlas al carrito. El sistema va mostrando los cambios al momento para que la compra sea fácil y rápida desde cualquier dispositivo.
![ropoa](https://github.com/user-attachments/assets/78ddfb8b-a287-4a2c-a3aa-fcf1dd1f39ae)


# requerimientos funcionales 
RF1: Ver las prendas disponibles con su precio.
RF2: Elegir la talla de la prenda.
RF3: Agregar prendas al carrito.
RF4: Quitar prendas del carrito.
RF5: Mostrar el precio total del carrito.

# requerimiento no funcionales 
 La tienda se puede usar en celular y computadora.
 La página se actualiza rápido sin recargar.
 El carrito se guarda aunque se cierre la página
 Los botones y textos son fáciles de entender


 # Tecnologias y Herramientas del ecosistema 
 Framework principal: React

Manejo de Estado: useState, ya que es una aplicación sencilla.

Consumo de Datos: Catálogo de productos desde un arreglo local.

Estilizado: CSS básico con Flexbox.

Despliegue: GitHub Pages o Netlify.
